## Installation

### Dependencies

- `git`
- `curl`
- `zsh`

### Install

```sh
$ git clone https://github.com/bitwolfe/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ zsh install
$ zsh # or just close and open your terminal again.
```
